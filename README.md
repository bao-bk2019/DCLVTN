# Welcome to Our Project!

# Install Enviroment
## For Mac
**Install environment:** 

`virtualenv env`

**Run environment:**

`source env/bin/activate`

## For Window

**Install environment:**

`python -m venv env`

**Run environment:**

`env/Scripts/activate`
# Install Packages

`pip install -r requirements.txt`

# Run Program

## Run Server
**Go to folder server:**

`python manage.py runserver`

## Run Client

**Install packages for Client**

`npm install`

**Run Client**

`npm start`
